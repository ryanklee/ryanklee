Hapax is a single-operator autonomous grounding lab: a live agentic system
studying grounded claim authority, refusal, correction, source provenance,
agentic oversight, privacy boundaries, and public egress under real operational
constraints.

The current research framing is:

> Grounding Attempts as Safety Artifacts: A Live Benchmark for Agentic Claim
> Confidence, Refusal, and Public Egress.

This is research infrastructure, not a product or community project. The useful
entry point is the dossier above, which maps the project to concrete repository
paths, public artifacts, and current evidence boundaries.
